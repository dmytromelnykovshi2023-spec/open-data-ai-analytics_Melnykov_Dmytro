Назва: Аналіз прозорості та прогнозування цінових аномалій у системі публічних закупівель Prozorro за допомогою методів машинного навчання.

Мета: Дослідження структури державних витрат через API Prozorro, виявлення статистичних закономірностей у тендерних пропозиціях та розробка моделі для автоматичного визначення аномальних цін або ризикових закупівель.

Джерело відкритих даних: Інформація про закупівлі, що оприлюднюється згідно із Законом України “Про публічні закупівлі” (data.gov.ua) — https://data.gov.ua/dataset/f9c0acdb-23c5-4a2d-b549-9868df8ca927.
